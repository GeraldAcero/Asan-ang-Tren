# Asan ang Tren?

Asan ang Tren? is a mobile-first route guide for commuters navigating Metro Manila's LRT-1, LRT-2, and MRT-3 lines.

The current web app is a React + Vite rebuild of the original site, updated with a cleaner trip planner, fare tools, dark mode, and a more app-like route experience.

---

## Features

- Search by station or landmark
  Choose a starting point and destination using station names or supported landmark names.

- Ticket-style trip planner
  A mobile-inspired planner UI for selecting origin and destination, swapping inputs, and using the nearest detected station.

- Journey timeline
  A step-by-step route view with line changes, stops, and transfer guidance.

- Fare calculator
  Estimate fares using SJT or SVC, passenger discounts, and Beep 2.0 simulation.

- Fare breakdown and history
  Review fare segments, compare fare variants, and keep a short local trip-spending history.

- Landmarks and nearby station context
  Open landmark details from stations to quickly see nearby destinations and context around a stop.

- Bookmarks and share links
  Save routes locally, reload them later, and generate shareable trip links.

- Dark mode
  Toggle between light and dark themes with styling tuned for both modes.

- Rotating commuter tips
  Helpful travel reminders and small guidance cues appear while using the planner.

---

## Who This Is For

This project is built for:

- first-time commuters learning how to move across Manila's rail lines
- tourists who are more familiar with landmarks than station names
- regular riders who want a quick route, fare estimate, and transfer summary
